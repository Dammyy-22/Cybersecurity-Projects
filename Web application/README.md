# Web Application Security

This folder contains projects and documentation on **web application security**, focusing on common vulnerabilities, testing tools, and mitigation strategies. It includes:

- **OWASP Top 10 exercises** (e.g., XSS, SQL Injection, Broken Access Control)  
- **Testing tools & scripts** (WebGoat, DVWA, Burp Suite)  
- **Reports & documentation** showing step-by-step analysis

## Purpose

Gain hands-on experience, understand vulnerabilities, and demonstrate secure web application practices.
